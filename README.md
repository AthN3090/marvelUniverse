# MARVELS UNIVERSE APP

MarvelsUniverse app is a website made in *React*, where you can find all your characters 
in a single place!.

## How to run
1. Clone the project
2. Move into the root dir
3. Run `npm install`
4. Project is running at `localhost:3000`
